About this "design system" page

This is possibly the most trivial and simple design system possible.

This page simply show-cases the default layout for pages in the app and the UI cards used to display information.

For the UI cards a few different cases are shown, when they have sub headers for tabular information and when they lack items or information.

Having this design system page enables me as a developer to try out and verify changes to the layout and styling of the app, directly in the app.
